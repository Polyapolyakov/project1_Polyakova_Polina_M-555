# project1_Polyakova_Polina_M-555
Текстовая adventure игра "Лабиринт сокровищ"

# описание
Игроку представлена консольная текстовая игрования игра, в рамках которой необходимо перемещаться между комнатами, собирать предметы, отгадывать загадки и, как итог, добраться до комнаты с сундоком с сокровищами и забрать эти сокровища.

# установка зависимостей:
make install

# запуск игры:
make project

# исследование:
Игроку в рамках задачи по поиску сокровища необходимо исследовать следующие комнаты:
1) entrance - вход в лабиринт;
2) hall - большой зал;
3) trap_room - комната с "изюменкой";
4) library - библиотека;
5) armory - оружейная;
6) treasure_room - комната с сундоком с сокровищами;
7) kitchen_room - кохонная комната;
8) bedroom - заброшенная спальная комната.

# цели:
Игроку необходимо перемещаться между комнатами, что бы добраться до комнаты с сундуком, для комнат предусмотрены предметы - "помошники", которые необходимо собирать, загадки для развлекательной цели, а также ряд "опасностей", которые игроку необходимо преодолеть.

# направления движения:
Для обнаружения комнаты с сокровищами игроку доступны четыре направления перемещения между комнатами: west, south, east и north.

# помощь:
Для успешного прохождения игроку доступна функция - помошник help, при выборе которой можно получить более детальную информцию.
